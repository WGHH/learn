## 兼容性

```
input {
	width:300px;
}
input::-webkit-input-placeholder {
	/* WebKit browsers */ 
    font-size:14px;
	color:red;
}
input:-moz-placeholder {
	/* Mozilla Firefox 4 to 18 */ 
font-size:14px;
	color:red;
}
input::-moz-placeholder {
	/* Mozilla Firefox 19+ */ 
font-size:14px;
	color:red;
}
input:-ms-input-placeholder {
	/* Internet Explorer 10+ */ 
font-size:14px;
	color:red;
}

```
